# openpopscr
Fits Jolly-Seber spatial capture-recapture by maximum likelihood

## Install
```
library(devtools)
devtools::install_github("r-glennie/openpopscr", build_vignettes = TRUE)
```

## Help 
A vignette describing how to use the package is included: 
```
library(openpopscr)
vignette("openpopscr") 
```
